How to use

Note: If you haven't used the Metafizzy Isotope Plugin yet, I would recommend playing with it first before using this plugin, given it's a hard dependency.

Inspect the markup on this demo to replicate filter setup (look for the class named "filtersnav"). Modify if desired, but if so, be sure to update the default class names in the options (see below).

In filter markup, for each filter (checkbox), in this demo named "filtersnav__filter", update the "data-filter" attribute to the name of the class you will use for the filtered items, then for each filtered item, add the class name for the corresponding data-filter (same as you would using the standard isotope plugin).

Set the js up as you normally would using the Metafizzy Isotope Plugin, then chain on ".isotopeDropdownFilters". 


  <script>
    $(document).ready(function(){
      var $container = $('.item-container')
      $container.isotope({
        itemSelector: '.item',
        layoutMode: 'fitRows' 
      }).isotopeDropdownFilters();
    })
  </script>

Options

Reference the markup in this demo to better understand options.

Most of the options reflect the markup of the filters.


  <script>
   
    isotopeDropdownFilters({
      container:            '.item-container' // container of filtered items
      filtersnav__filter:   '.filtersnav__filter' // individual filters (check boxes)
      filtersnav__item:     '.filtersnav__dropdown-item' // container of individual filter
      filtersnav__dropdown: '.filtersnav__dropdown' // filter group name
      filter__children:     '.filtersnav__dropdown-items' // parent of filter group
    });
    
  </script>
